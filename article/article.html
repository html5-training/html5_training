記事風ページ(仮)

HTML5のタグをまとめたページです。
普段の使用頻度が高くないものを中心に、自分が気になったものを抜粋しました。 ■ 参考サイト - TAG index

<pre>

ソースコードを整形済みテキストとして表示する際に使用。
＜ と ＞ は特殊文字扱いになるので、それぞれ &lt; と &gt; に置換する必要がある。

■ サンプルコード

<pre>&lt;strong&gt;健康ミネラル麦茶&lt;/strong&gt;
ミネラルバランスをととのえるため、&lt;i&gt;飲用海洋深層水&lt;/i&gt;を少量加えています。</pre>

■ ブラウザ表示

<strong>健康ミネラル麦茶</strong>
ミネラルバランスをととのえるため、<i>飲用海洋深層水</i>を少量加えています。

<blockquote>

「段落」単位で引用する際に使用。
フレーズ単位での引用は、q要素を使用する。
引用元の情報をblockquote内に配置する場合は、cite要素 または footer要素で囲う必要がある。

■ サンプルコード

<blockquote>
このページのトピックスは右のサイトを参考にまとめています - <cite><a>TAG index</a></cite>
</blockquote>

■ ブラウザ表示

このページのトピックスは右のサイトを参考にまとめています - TAG index

<hr>

article内におけるテーマ(話題)の区切りで使用。
段落単位の区切りで使うもので、「セクション間の区切り線」として配置するのは不適切。
※上下に0.5emのmarginが入り、線の太さや色はブラウザ間で若干違いがある。
※閉じタグなし

■ サンプルコード

<hr>

■ ブラウザ表示

<ol>

順序付きのリストをつくる際に使用。
リスト内の項目はli要素で表す。

■ サンプルコード

<ol>
<li>リスト項目1</li>
<li>リスト項目2</li>
<li>リスト項目3</li>
<li>リスト項目4</li>
</ol>

■ ブラウザ表示

  1. リスト項目1
  2. リスト項目2
  3. リスト項目3
  4. リスト項目4

<dl>

名前（dt要素）と値（dd要素）のグループで構成される記述リストを表す。

■ サンプルコード

<dl>
<dt>名前</dt>
<dd>名前に対する値</dd>
</dl>

■ ブラウザ表示

スマートフォン
高機能な携帯電話のこと。スマホと略されることもある。

<dt>

記述リスト（dl要素）における名前部分を表す。
dt要素はdd要素（値）とセットで使用。
dt要素の内容は1つのdl要素内で重複させない。

■ サンプルコード

<dl>
<dt>名前</dt>
<dd>名前に対する値</dd>
</dl>

■ ブラウザ表示

編集者
AAA
BBB

<dd>

記述リスト（dl要素）における値部分を表す。
dd要素はdt要素（名前）とセットで使用。
1つの用語（名前）に対して複数の説明（値）がある場合、dt要素の後に複数のdd要素を配置。また、複数の段落で構成された1つの説明文の場合、1つのdd要素内に記述。

■ サンプルコード

<dl>
<dt>名前</dt>
<dd>名前に対する値</dd>
</dl>

■ ブラウザ表示

やさしい麦茶
650ml
清涼飲料水
サントリーフーズ

<figure>

本文から参照される図版（挿絵、図表、写真、ソースコード等）を表す。
figure要素は離れた場所に配置した場合でも、本文には影響がない内容で使用可能。

■ サンプルコード

<figure>
参照される図版
</figure>

■ ブラウザ表示

<main>

文章のメインコンテンツを表す。
サイト内で共通使用されるグローバルナビゲーションは配置不可 ※ページ内リンク等のローカルナビゲーションであれば配置可能

■ サンプルコード

<main>
メインコンテンツ
</main>

■ ブラウザ表示

メインコンテンツ

記事その1

...

記事その2

...

記事その3

...

<cite>

作品のタイトル、著者、URLを表す際に使用。
引用元を示す際にもcite要素の使用が可能。
※デフォルトはイタリック体

■ サンプルコード

<cite>
作品のタイトル、著者、URL
</cite>

■ ブラウザ表示

新しい作品をアップロードしました！

https://www.youtube.com/watch?***

<q>

フレーズ単位で引用する際に使用。（段落単位はblockquote）
一般的なブラウザでは、q要素の前後に引用符がつけられるため自分で入力はしない。スタイルシートで指定することも可能。

■ サンプルコード

<q>
引用部分
</q>

■ ブラウザ表示

引用部分を表示する。
引用の中で引用部分を表示する。

<dfn>

用語の意味を説明する際に、その用語の部分に対してこの要素を使用。
dfn要素を使用する場合、dfn要素の親となる段落かセクション、または記述リストの同じグループ内にその用語に対する定義も含めなければならない。
定義を伴わない用語の場合は、i要素やb要素等を状況に合わせて使用する。

■ サンプルコード

<dfn>
定義される用語
</dfn>

■ ブラウザ表示

HTML
Webページを作成するためのマークアップ言語です。

<abbr>

略語や頭字語を表す際に使用。
abbr要素にtitle属性を指定すると、略される前の正式名称を示すことが可能。
※多くの人が知っているであろう略語の場合は、無理にabbr要素を使う必要はない。

■ サンプルコード

<abbr>
略語や頭字語を表す
</abbr>

■ ブラウザ表示

HTML

<sub>

下付き文字を表す際に使用。
sub要素は、下付き文字に特定の意味（数式や化学式等）がある場合のみ使用可能。

■ サンプルコード

<sub>下付き文字</sub>

■ ブラウザ表示

水の化学式はH2Oです。

<sup>

上付き文字を表す際に使用。
sup要素は、上付き文字に特定の意味（数式や化学式等）がある場合のみ使用可能。

■ サンプルコード

<sup>上付き文字</sup>

■ ブラウザ表示

質量とエネルギーの関係式：E=mc2

<time>

正確な日付や時刻を表します。
time要素は24時間表記の時刻、またはグレゴリオ暦による正確な日付を表す際に使用。

■ サンプルコード

<time>YYYY-MM-DDThh:mm:ssTZD</time>

■ ブラウザ表示

2017-08-08T13:05:30+09:00

<mark>

特定のテキストを目立たせたい場合に使用。
・例 ⇒ 検索ワードに一致する文字列、話題の対象になっているテキスト
※この要素はその文章における重要性や強調を表すものではなく、ユーザが参照しやすいように目立たせることが目的。

■ サンプルコード

<mark>ハイライト表示</mark>

■ ブラウザ表示

htmlの検索結果

ホームページの作成に役立つHTMLとCSSのリファレンス。TAG indexのHTMLリファレンスは ...

<ruby>

ルビ（ひらがな）をふるテキスト範囲を表す際に使用。
この要素内に対象テキスト（ルビベース）を配置して、rt要素でルビテキストを指定。

■ サンプルコード

<ruby>
例<rt>れい</rt>
題<rt>だい</rt>
</ruby>

■ ブラウザ表示

私の名前は 山田やまだ 太郎たろう です。

<rt>

対象テキスト（ルビベース）にルビテキストを表す際に使用。
この要素はruby要素またはrtc要素の子要素として使用。

■ サンプルコード

<ruby>
例<rt>れい</rt>
題<rt>だい</rt>
</ruby>

■ ブラウザ表示

私の名前は 山田やまだ 太郎たろう です。

<rtc>

ルビテキストのコンテナを表す。
この要素はruby要素の子要素として使用。
※rtc要素はルビベースに対して2種類のルビをふりたい場合に使用。(firefoxのみ有効？)

■ サンプルコード

<ruby>
例<rt>れい</rt>
題<rt>だい</rt>
<rtc>example</rtc>
</ruby>

■ ブラウザ表示

朝あさ寝ね坊ぼう Oversleeping

<rp>

ルビテキストを囲む括弧記号などを表す。
この要素はruby要素の子要素として使用。
rp要素の内容は、ルビ未対応のブラウザで表示される。（対応ブラウザでは非表示）

■ サンプルコード

<ruby>
例<rp>(</rp><rt>れい</rt>
題<rt>だい</rt>

</ruby>

■ ブラウザ表示

私の名前は 山田（やまだ） 太郎（たろう） です。

<strong>

テキストの強い重要性を表す。
strong要素を入れ子にすることで、重要性の度合いを高めることが可能。

■ サンプルコード

<strong>重要性を表したいテキスト</strong>

■ ブラウザ表示

重要なテキストの中でより強い重要性を表しています。

<span>

特定の範囲（段落内で使用するようなコンテンツフレージング・コンテンツ）を1つのグループとしてまとめる。
この要素自体は特別な意味は持たず、class属性、id属性、lang属性などと併せて使用。

■ サンプルコード

<p>緑色は、英語では<span lang="en">Green</span>、英語では<span lang="it">Verde</span>と表記します。</p>

■ ブラウザ表示

緑色は、英語ではGreen、イタリア語ではVerdeと表記します。

<br>

詩や住所などのように、改行がコンテンツの一部を形成する場合に使用可能。
改段落となる場合はp要素を使用。

■ サンプルコード

<p>〒000-0000<br>東京都千代田区X-X-XX<br>hogehogeビル 10F</p>

■ ブラウザ表示

〒000-0000
東京都千代田区X-X-XX
hogehogeビル 10F

<wbr>

折り返しが行われないテキスト（英単語）に対して、改行の機会を与える。

■ サンプルコード

☆wbrタグなし

<p>LongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWord</p>

★wbrタグあり

<p>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr>LongWord<wbr></p>

■ ブラウザ表示

☆wbrタグなし

LongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWord

★wbrタグあり

LongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWordLongWord

<ins>

その部分が後から追加されたものであることを明示したい場合に使用。

■ サンプルコード

<ul>
<li>リスト項目</li>
<li>追加されたリスト項目2</li>
</ul>

■ ブラウザ表示

  • リスト項目
  • 追加されたリスト項目

<del>

その部分が後から削除されたものであることを明示したい場合に使用。

■ サンプルコード

<p><del>削除されたテキスト。</del>一般のテキスト。</p>

■ ブラウザ表示

削除されたテキスト。一般のテキスト。

<picture>

レスポンシブイメージを文章内に埋め込む。
サイズや内容の異なる複数の画像候補を用意しておくことで、ユーザの閲覧環境に適した画像が表示される。
※picture要素内には、img要素の配置が必須（source要素の後に置く）。レスポンシブイメージに対応していないブラウザではこのimg要素のsrc属性で指定した画像が表示される。

■ サンプルコード

<picture>
<source media="(min-width: 1280px)" srcset="large.jpg" />
<img src="small.jpg" alt="サンプル">
</picture>

■ ブラウザ表示

<textarea>

複数行のテキスト入力欄を作成。
詳しいオプションはここを参照

■ サンプルコード

<textarea name="example1"></textarea>

■ ブラウザ表示

<button>

3タイプのボタン（送信submit、リセットreset、汎用button）を作成。
※input要素のボタンとは異なり、button要素では内容を持てる（ボタン上に表示される）。また、画像をボタンのラベルとして表示することも可能。
詳しいオプションはここを参照

■ サンプルコード

<button type="submit" name="example" value="ボタン">ボタンの内容</button>

■ ブラウザ表示

<form>

入力フォームを作成。この要素内にフォームを構成するパーツを配置。
通常、フォームを構成するパーツ（入力欄や送信ボタン等）はform要素内に配置する。
詳しいオプションはここを参照

■ サンプルコード

<form></form>
